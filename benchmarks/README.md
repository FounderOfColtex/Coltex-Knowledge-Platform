# Benchmark Datasets

Curated evaluation datasets generated by `make benchmarks` or `make product`.

| File | Description |
|------|-------------|
| `faq_pairs.jsonl` | Question → expected document ID pairs |
| `retrieval_gold.jsonl` | Retrieval queries with relevant doc IDs |
| `rag_eval.jsonl` | RAG evaluation rubric with expected keywords |
| `evaluation_report.json` | Evidence report from `make evaluate` |
| `benchmark_manifest.json` | Dataset counts and metadata |

Run evaluation:

```bash
make evaluate
```

See [docs/product-evaluation.md](../docs/product-evaluation.md) for metric definitions.
